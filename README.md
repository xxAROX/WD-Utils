### What does this plugin?
- You have the positions of the players
- You can play event sounds via api to a player
- [`ScriptCustomEventPacket`](./docs/ScriptCustomEventPacket.md)
- `/wdreload` command (**DO NOT USE IN PRODUCTION**)
- `/wdplugins` command
